改造体重秤

Weight Scale Hack

Download this project as a .zip file Download this project as a tar.gz file

项目简介：

这个项目的来源是因为我想把身边的普通体重秤变得能加智能，能够记录自动记录数据，使得用户能够直观的看到自己体重变化的趋势，对自己的健康状况更加了解。在本地记录数据不是很可靠，而且显示数据也是很大的问题，于是采用的方法是把数据传到网上，用服务器进行数据的记录，分析和显示。 为了实现这个效果，基本思路是每一次用户称重的时候，体重秤就会自动把数据传到远程的服务器保存下来。用户可以通过电脑或者智能手机看到直观的体重变化曲线。

第一步，需要先把体重的时候读出来。买了一个亚马逊排名第一的体重计,先研究一下里边的原理。
